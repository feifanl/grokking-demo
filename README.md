# grokking-demo
A foray into mechanistic interpretability using the case of a 1-layer transformer grokking modular addition -- techniques include principal component analysis, discrete Fourier transforms, ablation, linear probing, and activation patching 

The accompanying blog post can be found [here]([url](https://open.substack.com/pub/feifan2/p/deconstructing-grokking-how-a-tiny?utm_campaign=post-expanded-share&utm_medium=post%20viewer)).
